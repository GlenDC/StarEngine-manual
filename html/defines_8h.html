Enumeration Type Documentation

enum DirectoryMode : byte
strong

Enumeration that describes where the root of the path of a file can be found.

Enumerator
assets 

This is the default Assets folder on Android. On windows this directory is defined in the Win32Manifest.xml

internal 

This is the default APK internal folder on Android. On windows this directory is defined in the Win32Manifest.xml

external 

On android the root directory is '<SD-CARD>/<GAME-NAME>/'. On windows the root directory is '???/documents/<GAME-NAME>/'.

custom 

Using this mode gives you the freedom to define a dynamic or static path yourself. We advice that you never use a static path in this mode, as it isn't portable to another device and/or platform.
